2. 解压 / 安装

Android 安装

下载 APK 后，打开文件进行安装。如果系统提示“禁止安装未知来源应用”，请在系统设置里允许浏览器或文件管理器安装应用。

导入订阅

打开 Clash Meta 后，在配置页面导入你的订阅链接，然后选择节点并启动代理。

Windows / iOS

Windows 和 iOS 版本目前未放置下载链接，后续可以继续添加到表格中。
