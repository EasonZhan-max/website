2. 登录并下载

03输入上方获取的共享账号和密码，点击「登录」
04如弹出「Apple ID 安全」提示，点击「其他选项」，再选择「不升级」
05搜索 Shadowrocket 并下载安装。
06下载完成后，回到头像页面退出共享账号，重新登录自己的 Apple ID
