# Eason 的中转站 - 美化版软件下载站

## 文件结构

```txt
index.html
style.css
script.js
software/
└─ clash-meta.html
```

## 功能

- 首页软件分类
- Clash Meta 独立下载页
- 深色 / 浅色模式切换
- 搜索软件分类
- 下载表格
- 项目入口
- 子域名导航
- 手机端适配

## GitHub Pages 部署

把所有文件上传到仓库根目录，开启 GitHub Pages 即可。

## 修改下载链接

打开：

```txt
software/clash-meta.html
```

修改表格里的 `<a href="...">点击下载</a>` 链接即可。
