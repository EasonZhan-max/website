如何选择版本

不同平台和不同架构版本的简单说明。

Android

大多数人不知道设备架构时，优先下载 Universal 通用版；如果你确认是新手机，通常也可以选择 arm64-v8a。

Windows / macOS

桌面端通常按系统平台区分；macOS 还可能区分 Apple Silicon 和 Intel。

iOS

iOS 通常通过 App Store、TestFlight 或付费客户端获取，不是通过 APK 安装。
